# React Task Manager

A task management app built with React that allows users to organize tasks with filtering, priorities, due dates, and dark mode.

## Features:

- Add/edit/delete tasks
- Task completion tracking
- Due dates
- Overdue indicators
- Dark mode
- Search/filter/sort
- Local storage persistence

## Built With

- React
- JavaScript
- CSS
- Git/GitHub
